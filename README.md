# Personal Portfolio Website 🌐

A simple personal portfolio website created using **HTML and CSS** to showcase my profile, skills, interests, projects, and contact information.

## 📌 About the Project

This project contains a Home page and a Contact page. The Home page introduces me as a FullStack Developer and Web 3.0 enthusiast, along with my interests and projects. The Contact page provides a simple form for visitors to connect with me.

The website is designed with a clean layout and custom CSS styling to create an attractive and user-friendly interface.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts

## ✨ Features

- Personal introduction section
- About Me section
- Interests showcase
- Projects section
- Contact page with form
- Social media links
- Custom CSS design
